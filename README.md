# 🧁 Naliya Bites

**Naliya Bites** adalah website landing page untuk bisnis homemade brownies premium dengan berbagai pilihan topping favorit. Dibuat dari bahan berkualitas, fresh setiap hari — cocok untuk hampers, party, dan camilan spesial.

---

## ✨ Fitur

- Landing page yang responsif dan modern
- Animasi halus menggunakan **Framer Motion**
- Desain elegan dengan tipografi **Playfair Display**, **Poppins**, dan **Nunito**
- Tampilan produk, menu, dan informasi pemesanan
- Dioptimalkan untuk mobile dan desktop

## 🚀 Cara Menjalankan

### Prerequisites

Pastikan sudah menginstall **Node.js** dan **npm** di komputermu.

### Instalasi

```bash
# Clone repository ini
git clone https://github.com/alhilalfathi/naliya-bites.git

# Masuk ke direktori project
cd naliya-bites

# Install dependencies
npm install
```

### Development

```bash
# Jalankan development server
npm run dev
```

Buka browser dan akses `http://localhost:5173`

### Build Production

```bash
# Build untuk production
npm run build

# Preview hasil build
npm run preview
```

## 🎨 Desain
